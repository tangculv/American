### 支持的筛选参数

#### 市值相关
- `market_cap_more_than` - 最小市值（单位：百万）
- `market_cap_less_than` - 最大市值
- `market_capitalization_more_than` - 大于某市值

#### 估值相关
- `pe_less_than` - PE小于某值
- `pe_more_than` - PE大于某值
- `pe_between` - PE在两个值之间
- `pb_less_than` - PB小于某值

#### 财务相关
- `eps_greater_than` - EPS大于某值
- `eps_less_than` - EPS小于某值
- `dividend_yield_more_than` - 股息率大于某值
- `dividend_yield_less_than` - 股息率小于某值
- `free_cash_flow_more_than` - 自由现金流大于某值

#### 行业/交易所
- `sector` - 行业代码（GICS行业）
- `exchange` - 交易所代码（NYSE、NASDAQ、AMEX）
- `is_etf` - 排除ETF
- `is_fund` - 排除基金

#### 技术指标
- `beta_more_than` - Beta大于某值
- `beta_less_than` - Beta小于某值

#### 其他
- `limit` - 返回结果数量（默认20）
- `offset` - 分页偏移量
- `order` - 排序方式（asc、desc）

### API端点

#### 筛选端点
```
GET /v3/stock/market/screener
```

**请求示例**：
```python
import requests

api_key = "YOUR_API_KEY"

params = {
    "market_capitalization_more_than": 500000000,  # >5亿美元
    "pe_less_than": 25,                      # PE < 25
    "is_etf": false,                            # 排除ETF
    "is_fund": false,                           # 排除基金
    "is_actively_trading": true,              # 活跃交易
    "limit": 20,                                 # 返回20只
    "apikey": api_key
}

response = requests.get(
    "https://financialmodelingprep.com/api/v3/stock/market/screener",
    params=params
)

data = response.json()
# 返回20只符合条件的股票列表
```
